# TodoList API

Projeto desenvolvido durante o **Minicurso de Java da Rocketseat**, com foco na construção de uma aplicação back-end utilizando **Java e Spring Boot**.

A aplicação consiste em uma API para gerenciamento de tarefas (To-Do List), permitindo criar, visualizar, atualizar e remover tarefas.

---

## 🚀 Tecnologias utilizadas

- Java
- Spring Boot
- Maven
- Lombok
- H2 Database
- REST API

---

## ⚙️ Funcionalidades

- Criar uma tarefa
- Listar tarefas
- Atualizar tarefas
- Remover tarefas
- Persistência de dados em banco

---

## 📂 Estrutura do projeto

A aplicação segue uma arquitetura comum em aplicações Spring Boot:

- **Controller** → responsável pelas rotas da API  
- **Service** → lógica de negócio  
- **Repository** → acesso ao banco de dados  
- **Model** → entidades da aplicação
